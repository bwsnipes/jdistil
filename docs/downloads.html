Framework Source
The JDistil source code is available at the following repository link. This includes source for the core library and the codes and security module libraries.

https://github.com/bwsnipes/jdistil
Eclipse Plugin
The JDistil plugin can be downloaded and installed into Eclipse using the following update site. The plugin contains all JDistil framework libraries and these libraries will be included in any project created using the plugin.

http://www.jdistil.com/plugin/updates
